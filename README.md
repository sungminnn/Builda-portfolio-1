# LUNE Interior Studio Portfolio

A premium interior design studio website built with Next.js, React, and TypeScript. The project presents a polished brand experience for a fictional high-end studio, combining full-screen visual storytelling, animated section reveals, responsive layouts, and a consultation-oriented contact flow.

## Highlights

- Built a single-page portfolio experience with Next.js App Router and React 19.
- Implemented a rotating hero image system, scroll-triggered reveal animations, and image-heavy service sections.
- Designed a luxury editorial layout with responsive navigation, portfolio/service sections, CTA blocks, and a structured contact form.
- Used `next/image` for optimized image rendering and layout-stable visual assets.
- Kept the implementation lightweight with no CMS or backend dependency, making the site easy to deploy as a static marketing experience.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- CSS Modules / Global CSS
- ESLint

## Project Structure

```text
app/
  globals.css   # Global styling and responsive layout rules
  layout.tsx    # App metadata and root layout
  page.tsx      # Main landing page experience
public/
  images/       # Portfolio and hero imagery
```

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Overview

The project focuses on a brand-driven landing page experience with strong visual hierarchy, responsive composition, animation timing, optimized media handling, and a consultation-oriented page structure.
